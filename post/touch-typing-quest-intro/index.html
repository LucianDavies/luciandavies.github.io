Touch Typing Quest (Intro)

June 3, 2020

Since the beginning of time I have always wanted to type really fast but with so many things to do the benefits haven’t always been clear. In the spirit of learning new things I am to attempt this quest again but this time document my efforts. As a software developer strong typing skills can be an assets despite software development being short bursts of typing instead of the long stream we see in other professions.

To add a bit of fun to this I am going to structure this quest into 4 section:

  • Software
  • Hardware
  • Practice Patterns
  • Conclusion

All this said and done I would like to see me have a decent word count 40 Words Per Minute (WPM) using all 10 fingers from an unknown word count using 6 fingers. To try be fair and also give us lots of typing exercises to evaluate the software and keyboards there is going to be a lot of typing and retyping. Maximum effort.
